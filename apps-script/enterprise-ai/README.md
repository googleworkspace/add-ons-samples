# Enterprise AI Google Workspace add-on

**Note:** This project is part of the official Google Codelab [Integrate Gemini Enterprise Agents with Google Workspace](https://codelabs.developers.google.com/ge-gws-agents).

Integrates with the *default* Gemini Enterprise AI agent natively.

## Supported Applications

- **Gmail**: Adds a contextual sidebar to the inbox. When an email is selected, the Add-on extracts its subject and body natively and passes it as context to the Gemini Enterprise AI model, allowing users to ask questions in its specific contect.

- **Google Chat**: Adds a chat app integration allowing users to converse directly with the Gemini Enterprise Agent natively within a Chat DM space.
